# MuHub — ผูกดวงโหราศาสตร์ไทย

แอปพลิเคชันผูกดวงโหราศาสตร์ไทยระบบนิรายนะ (ลาหิรี) แสดงผลผ่านหน้าแดชบอร์ดระดับพรีเมียมแบบ 3 คอลัมน์ (ภาพดวงชะตาจักราศี, ตารางสมผุสดาว, และตารางทักษาจร) พร้อมระบบฐานข้อมูลตำแหน่งดาวจริงช่วงปี พ.ศ. 2443 - 2743 (ค.ศ. 1900 - 2200)

## ฟีเจอร์หลัก (Key Features)

1. **แดชบอร์ด 3 คอลัมน์พรีเมียม:** แสดงผลพร้อมกันทั้งดวงชะตา (Chakra Wheel), ตารางแสดงค่าสมผุส และตารางทักษาจร ในหน้าจอเดียว
2. **ระบบดวงจร & ลัคนาอายุจร:** แสดงผลลัคนาอายุจรเป็นรูปดาวสีเขียว (★) ที่วงนอก (ดวงจร) โดยคำนวณและขยับตำแหน่งหลบดาวจรดวงอื่น ๆ อัตโนมัติเพื่อป้องกันการซ้อนทับกัน
3. **ระบบฐานข้อมูล Ephemeris (ลาหิรี):** โหลดข้อมูลสมผุสดาวจริงจากไฟล์ฐานข้อมูล CSV ช่วงปี 1900 - 2200 ด้วยอัตราการโหลดที่รวดเร็ว พร้อมแสดงผลตำแหน่งดาวเคราะห์รอบนอก (มฤตยู, เนปจูน, พลูโต) และเกตุไทย (๙)
4. **โหมดลากลางและไม่ทราบเวลาเกิด:** สามารถระบุเวลาเกิดที่ไม่ทราบได้ (ระบบจะล็อคที่เวลา 06:00 น. โดยอัตโนมัติตามหลักโหราศาสตร์)
5. **ระบบคำนวณสำรอง (Browser Fallback):** หากไม่ได้ใช้งาน API หรือฐานข้อมูลชิ้นใหญ่ไม่พร้อม ระบบจะคำนวณแบบสถิติผ่านเบราว์เซอร์อัตโนมัติทันทีโดยไม่รบกวนการใช้งานของผู้อื่น

## โครงสร้างไฟล์สำหรับอัปโหลดขึ้น GitHub (Repository Structure)

เมื่อต้องการจะเผยแพร่แอปพลิเคชันนี้ขึ้น GitHub แนะนำให้อัปโหลดไฟล์หลักเหล่านี้ขึ้นไป (ไฟล์ขนาดใหญ่และระบบแคชได้รับการข้ามผ่านไฟล์ `.gitignore` แล้ว):

*   **`index.html`** (หน้าเว็บแอปพลิเคชันฝั่งผู้ใช้งานหลัก)
*   **`astronomy.browser.min.js`** (ไลบรารีคำนวณดาราศาสตร์สำรองบนเบราว์เซอร์)
*   **`logo.png`** (โลโก้ตราสัญลักษณ์อย่างเป็นทางการของ MuHub)
*   **`server.py`** (FastAPI Web Server และ API สำหรับดึงพิกัดดวงดาวจากฐานข้อมูล)
*   **`start-server-python.bat`** (สคริปต์แบบดับเบิ้ลคลิกสำหรับเข้าใช้งานบน Windows)
*   **`thai-astrology-ephemeris/thai_ephemeris_1900_2200.zip`** (ไฟล์ฐานข้อมูลแบบบีบอัดขนาด ~31MB ซึ่งจะถูกขยายไฟล์เป็น CSV อัตโนมัติเมื่อรันเซิร์ฟเวอร์ครั้งแรก โดยไม่เกินขีดจำกัดไฟล์ 100MB ของ GitHub)
*   **`.gitignore`** (ไฟล์ละเว้นการจัดส่งไฟล์ขนาดใหญ่อย่าง JPL Ephemeris `de440.bsp` และตัว CSV ดิบ 242MB ขึ้นระบบ Git)
*   **`README.md`** (ไฟล์เอกสารคำแนะนำฉบับนี้)

## ขั้นตอนการติดตั้งและเริ่มใช้งาน (Getting Started)

### 1. ความต้องการของระบบ (Prerequisites)
*   ติดตั้ง **Python 3.x**
*   แนะนำให้ติดตั้งระบบจัดการแพ็คเกจแบบรวดเร็วพิเศษ **`uv`** (หากไม่มี ระบบจะใช้ตัวจัดการปกติของเครื่องได้)

### 2. วิธีการรันบน Windows (Easy Launch)
1. ดับเบิ้ลคลิกเปิดใช้งานไฟล์ **`start-server-python.bat`**
2. ระบบจะเริ่มตรวจหาไฟล์ฐานข้อมูล หากยังไม่เคยเปิดมาก่อน ระบบจะทำการแตกไฟล์จาก `thai_ephemeris_1900_2200.zip` ให้โดยอัตโนมัติ
3. เปิดเบราว์เซอร์ไปที่ลิงก์: **`http://localhost:3737/`** เพื่อใช้งาน

### 3. วิธีรันเซิร์ฟเวอร์แบบ Manual (Cross-Platform)
คุณสามารถรันผ่านเทอร์มินัล/CMD ได้โดยพิมพ์คำสั่ง:
```bash
uv run server.py
# หรือหากใช้ Python ปกติในการลง dependencies:
pip install fastapi uvicorn
python server.py
```
